Introduction
Floof Station is a powerful instrument plugin that hosts third-party instrument, audio effect, and MIDI effect plugins. In short, it lets you combine your favorite plugins into one unified instrument. Whether you're building layered soundscapes or complex processing chains, Floof Station is designed to make advanced setups intuitive and fast.
Scanning for Plugins
To begin using your plugins, you'll first need to scan your system.
  1. Click the burger button at the bottom right to open the settings menu.
  2. Click the Plugin Manager button to open it in a separate window.
  3. Click Scan Plugins and follow the on-screen instructions to locate your plugin folders.
Once the scan is complete, all compatible plugins will be available within Floof Station.
Loading an Instrument into a Rack
  1. Click the menu on a rack unit.
  2. Choose Load Plugin, then select from your available plugins.
The plugin will load in a few seconds and will be ready to play. You can load another instrument in the rack unit below and stack them together. To open a plugin’s interface, click its name in the rack unit.
Paw Pages
Each rack unit has three functional pages, accessible via the paw icon:
  • Audio Page
  • MIDI Page
  • Macro Page
Audio-related controls are green, while MIDI-related controls are blue. This color scheme is consistent throughout the plugin. You can also use the global paw button in the bottom bar to switch pages for all rack units simultaneously.
Macro Page
The Macro page includes six assignable macro controllers.
  1. Right-click a macro controller and select Assign.
  2. The controller will blink. Move the desired parameter in the plugin's user interface.
Macros can be automated from your DAW or host, allowing expressive control and easy modulation.
Audio Page
The Audio page includes five audio sends labeled A–E.
Ctrl+Click (Cmd+Click on macOS) a send controller to toggle between pre-fader and post-fader mode.

The sixth controller sets the main audio output, with the following options:
  • Master – Sends audio directly to the DAW/host.
  • None – Disables direct output; however, audio sends will still function.
  • Bus 1–12 – Routes audio to one of 12 buses, recognized as separate channels in your DAW/host for multi-output setups.
MIDI Page
The MIDI page offers advanced filtering and routing options:
  • Note range filter: Set a note range where the unit will respond
  • Velocity filter: Let the plugin only respond within this velocity range
  • Note offset: Offset the notes going into the plugin
  • Octave offset: Offset the notes going into the plugin by octaves
  • MIDI channel filter: Filter only certain MIDI channels into the plugin
The last knob determines the MIDI output target, allowing you to send MIDI to other units, no units or to the DAW/host.
Audio & MIDI Receive
Each plugin can receive audio and MIDI from the following sources:
  • Host
  • Unit Above (in the rack)
  • None
  • Sends A–E in audio or Y–Å in MIDI
Selecting Unit Above is ideal for creating serial chains of audio and MIDI effects.
Rack Unit Management
  • Add a unit: Click the plus (+) button in the bottom bar.
  • Remove a unit: Click the rack’s menu button and select Remove Rack.
  • Move a unit: Drag it by clicking and holding an empty space on the unit.
Misc Controls
Located below the central display screen, you’ll find:
  • Solo / Mute
  • Level and Pan controls
  • MIDI Panic – Sends Note Off for all notes on the current unit. You may need to click it more than once if some notes remain stuck.
Window Modes
Each plugin’s GUI can be displayed in one of two ways:
  • Common Window – Shared among plugins
  • Individual Window – Separate for each plugin
Set the window mode by opening the rack unit’s menu and selecting your preferred mode. You can also set the default window mode in the settings section.
Presets
Save and recall complete rack configurations:
  • Click the save icon to open the Save As dialog.
  • Ctrl+Click the icon to quickly overwrite the current preset.
Themes
You can change themes per rack unit via its menu, or apply a global theme from the Settings section. Additionally, the color intensity can be adjusted in the Settings section.
INSTALLATION GUIDE
Windows
Installation of this plugin is as simply as unzipping the zip file to uncover a folder with a .vst3 ending. Place this entire folder in a directory where your DAW host is able to locate it.

VST3 plugins for Windows are typically installed in either "C:\Program Files\Common Files\VST3" or "C:\Program Files\VSTPlugins". However, the exact location can vary depending on the configuration of your DAW host.
Mac
Use the provided installer to install Floof Station on your Mac. The installer will place the plugin in the correct location for your DAW host to find it.
